Funny and fancy school project about a parody of Pokemon using anime Girl (Waifu)  

## Installation 
```bash
git clone https://github.com/YuToutCourt/WaifuMon.git
cd WaifuMon
pip install -r requirements.txt
```

## Usage

```bash
python3 main.py
```
